# Baggies App

Private app for Mt. Albert Ponsonby Baggies — a social football club in Auckland, NZ.

## What It Does
- Player availability & RSVP tracking
- Match results & team news
- Banter feed & match stories
- Player profiles & stats

## Contributing
- Always branch off `main`: `git checkout -b feature/your-feature`
    feature/   → new functionality       e.g. feature/player-profiles
    fix/       → bug fixes               e.g. fix/rsvp-not-saving
    chore/     → maintenance & config    e.g. chore/update-dependencies
    docs/      → documentation only      e.g. docs/update-readme
    test/      → test additions only     e.g. test/rsvp-spec-shells
- Write SpecKit test shells before writing feature code
- Open a PR for review before merging
