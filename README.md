# Baggies App

A private club app for Mt. Albert Ponsonby Baggies FC.

**Live prototype:** https://baggies-fc-app.vercel.app

## What it does

- Coach creates matches and players
- Players join via a club invite link and claim their player record
- Players RSVP availability for each match
- Coach builds and publishes the team sheet
- Players see the published team sheet

## Tech

- Vite + React + Tailwind CSS
- All data stored in the browser (localStorage) — no backend yet
- Deployed on Vercel

## Running locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173
